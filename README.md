#  AI Chatbot with Memory

A Python-based chatbot that remembers user interactions using persistent memory storage.

## Features
- Interactive chat system
- Memory of past conversations
- Persistent storage using JSON
- Modular and clean architecture

## Tech Stack
- Python
- File handling (JSON)
- Basic NLP logic

## Project Structure
- chatbot/ → core chatbot logic
- data/ → memory storage
- utils/ → helper functions

## Goal
To simulate a simple AI assistant with memory capabilities.
